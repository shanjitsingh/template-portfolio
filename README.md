# Andrea Colalleli | Corporate & Business Attorney

A professional portfolio website for Andrea Colalleli, a corporate and business attorney. Built with React and Vite.

## Features

- **Hero Section** - Professional introduction with call-to-action buttons
- **About Section** - Attorney biography with experience statistics
- **Practice Areas** - Grid display of legal services offered
- **Contact Section** - Contact information and message form

## Tech Stack

- React 18
- Vite 5
- CSS3 (no external UI libraries)

## Getting Started

1. Install dependencies:

```bash
npm install
```

2. Run the development server:

```bash
npm run dev
```

3. Build for production:

```bash
npm run build
npm run preview
```

## Customization

- Replace placeholder images in the Hero and About sections with actual attorney photos
- Update contact information in `src/components/Contact.jsx`
- Modify practice areas in `src/components/Services.jsx`
- Update biography text in `src/components/About.jsx`
- Adjust color scheme in `src/styles.css` using CSS variables

## Color Palette

The site uses a professional navy blue theme:
- Navy Dark: `#0d1b2a`
- Navy: `#1d3557`
- Navy Light: `#457b9d`
