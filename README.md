# Vittoria — Portfolio

This is a minimal Vite + React portfolio for Vittoria, a dark/emo tattoo artist.

Getting started:

1. Install dependencies

```bash
npm install
```

2. Run the dev server

```bash
npm run dev
```

3. Build for production

```bash
npm run build
npm run preview
```

Replace the sample SVGs in `src/assets/` with actual photos of work. Update contact details in `src/components/Contact.jsx`.
